# Firebase (Analytics + Crashlytics)

1) Create Firebase project and add Android app.
2) Download `google-services.json` into:
   `C:\rockfit\app\google-services.json`
3) Enable Analytics + Crashlytics in Firebase console.
4) Add Gradle plugins:
   - `com.google.gms.google-services`
   - `com.google.firebase.crashlytics`
5) Add dependencies:
   - `firebase-analytics`
   - `firebase-crashlytics`

This project is ready for wiring once the config file is available.
